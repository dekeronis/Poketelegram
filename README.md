# Telegram Pokémon Bot

*Небольшой* Telegram-бот для ~~ловли~~ тренировки и ухода за покемонами.

Бот написан на Python с использованием библиотеки `pyTelegramBotAPI`.

---

## Функциональность

- Создание покемона для пользователя 
    
- Просмотр информации о покемоне
    
- Тренировка покемона с получением опыта
    
- Кормление покемона для восстановления здоровья
    
- Разные типы покемонов: `Wizard` и `Fighter`, каждый со своими особенностями

---

## Запуск

1. Установи зависимости:
    

`pip install -r requirements.txt`

2. *Добавь токен* бота в `config.py`:
    

`token = "YOUR_TELEGRAM_BOT_TOKEN"`

3. Запусти бота:
    

`python main.py`

---

## Использование команд

|Команда|Описание|
|---|---|
|`/go`|Создать покемона|
|`/info`|Посмотреть характеристики своего покемона|
|`/train`|Тренировать покемона и получать опыт|
|`/feed`|Кормить покемона и восстанавливать здоровье|

---

## Особенности

- Редкость покемона влияет на бонусы (`common`, `rare`, `ultra`, `mythical`)
    
- Классы покемонов (`Wizard` и `Fighter`) имеют уникальные характеристики
    
- XP и уровни рассчитываются динамически
    
- Подключение к PokeAPI для получения имени, типа и изображения покемона
